How to use


The purpose of this website is to create a summary of your existing budget. Once your summary has been created, you will be able to see your total saving and an overview of your income and expenses. You can also convert these to different time frequencies, for example, if you pay $9.99/month for Netflix, easily change it to see how much that costs per year.

There are three sections

  • Income
  • Fixed Expenses
  • Variable Expenses


To add an income/expense to each section click this button
 Add



This box will then appear. First, add the name of the income/expense, then add the amount and finally the frequency e.g. Work, $40,000 per year. To add this, click the green tick, to cancel, click the red X.
